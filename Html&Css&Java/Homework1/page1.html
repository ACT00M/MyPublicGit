تعريف بالوظيفة

نبحث عن مصمم جرافيك مبدع وذو خبرة للانضمام إلى فريقنا. المهام تشمل تطوير وتصميم جرافيك.


شاهد الفيديو التعريفي

المهام المطلوبة

  1. تصميم الشعارات والهويات البصرية.
  2. إنشاء المواد التسويقية مثل البروشورات والملصقات.
  3. تصميم واجهات المستخدم للتطبيقات والمواقع.
  4. العمل مع فريق التسويق لتحقيق الأهداف المرجوة.
  5. استخدام أدوات التصميم الحديثة مثل Adobe Creative Suite.

الخبرات المطلوبة

  • خبرة في استخدام برامج التصميم مثل Photoshop وIllustrator.
  • فهم جيد لمبادئ التصميم الجرافيكي.
  • القدرة على العمل تحت الضغط وتلبية المواعيد النهائية.
  • خبرة في تصميم واجهات المستخدم (UI/UX) تعتبر ميزة إضافية.
